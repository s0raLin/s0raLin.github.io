| 运算符 | 对应的方法   |
| ------ | ------------ |
| a+b    | a.plus(b)    |
| a-b    | a.minus(b)   |
| a\*b   | a.times(b)   |
| a/b    | a.div(b)     |
| a%b    | a.rem(b)     |
| a..b   | a.rangeTo(b) |

示例

```kotlin
 fun main() {
     val a = 10
     val b = 3

     println(a + b)        // 运算符方式：13
     println(a.plus(b))    // 方法调用方式：13

     println(a - b)        // 7
     println(a.minus(b))   // 7

     println(a * b)        // 30
     println(a.times(b))   // 30

     println(a / b)        // 3
     println(a.div(b))     // 3

     println(a % b)        // 1
     println(a.rem(b))     // 1

     println(a..b)         // 10..3（注意这是一个空区间，因为 10 > 3）
     println(a.rangeTo(b)) // 10..3
 }

```
