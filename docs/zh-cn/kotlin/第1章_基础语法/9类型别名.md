如果类型名太长，我们可以使用较短的名称来代替原类型名
语法:
> typealias 类型别名 = 已有类型 </br>
示例:
```kotlin
 class A {
     inner class Inner
 }
 class B {
     inner class Inner
 }
 typealias AInner = A.Inner

 typealias BInner = B.Inner
 fun main() {
     // 使用AInner定义变量、对象
     val a: AInner = A().AInner
     //使用BInner定义变量、对象
     val b: BInner = B().BInner
 }
```
