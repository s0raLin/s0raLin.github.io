# 欢迎使用日记草稿箱

这是一篇示例日记，展示了日记草稿箱的功能。

## 功能说明

1. **新增日记** - 通过 API 创建新的日记草稿
2. **编辑日记** - 修改现有日记内容
3. **删除日记** - 删除不需要的草稿
4. **发布日记** - 将草稿移动到正式日记目录

## 可用的 Frontmatter 字段

- `title` - 日记标题
- `mood` - 心情状态
- `location` - 地点
- `tags` - 标签数组
- `images` - 图片数组
