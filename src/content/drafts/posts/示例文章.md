# 欢迎使用文章草稿箱

这是一篇示例文章，展示了草稿箱的功能。

## 功能说明

1. **新增草稿** - 通过 API 创建新的草稿
2. **编辑草稿** - 修改现有草稿内容
3. **删除草稿** - 删除不需要的草稿
4. **发布草稿** - 将草稿移动到正式文章目录

## 如何使用 API

### 获取草稿列表

```bash
GET /api/drafts.json
```

### 新增文章草稿

```bash
POST /api/drafts.json
Content-Type: application/json

{
  "type": "posts",
  "title": "我的新文章",
  "content": "文章内容...",
  "category": "技术",
  "tags": ["TypeScript"]
}
```

### 更新草稿

```bash
PUT /api/drafts.json
Content-Type: application/json

{
  "type": "posts",
  "id": "我的新文章",
  "title": "更新后的标题",
  "content": "更新后的内容..."
}
```

### 删除草稿

```bash
DELETE /api/drafts.json?type=posts&id=我的新文章
```
