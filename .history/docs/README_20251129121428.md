## 📖 学习内容

### Kotlin 教程
Kotlin 是一门现代的、静态类型的编程语言，由 JetBrains 开发。本教程从基础语法开始，逐步深入到高级特性，帮助你掌握 Kotlin 的核心概念和最佳实践。

- [开始学习 Kotlin](zh-cn/kotlin/README.md)
  - 第1章：基础语法
  - 第2章：运算符和表达式
  - 第3章：流程控制
  - 第4章：集合与数组
  - 第5章：函数和Lambda表达式

### Git 教程
Git 是分布式版本控制系统，是现代软件开发不可或缺的工具。本教程涵盖 Git 的基本概念、常用命令和高级用法。

- [开始学习 Git](zh-cn/git/README.md)
  - Git 入门
  - Git 内部结构
  - Git 创建仓库
